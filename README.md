# Wallet Management API (Laravel)

## Setup
1. Clone the repo
2. Run `composer install`
3. Copy `.env.example` to `.env` and update DB credentials
4. Run migrations: `php artisan migrate`
5. (Optional) Seed test data: `php artisan db:seed`
